# fish_sales
# Retail Promotion Effect Analysis & Sales Forecasting

Репозиторий содержит код и данные для исследования влияния промо-акций и политики скидок на объём продаж в розничных сетях на панельных данных (SKU × клиент × месяц).

Проект выполнен в рамках учебно-исследовательской работы и включает статистический анализ, машинное обучение и решение обратной задачи по подбору скидки для достижения целевого роста продаж.

---

## Структура репозитория

### Анализ данных
- **EDA1.py**  
  Статистический и разведочный анализ данных:
  - анализ распределений продаж и скидок;
  - исследование сезонности;
  - оценка зависимости продаж от промо-активности;
  - визуализация ключевых закономерностей.

---

### Моделирование и прогнозирование
- **models.py**  
  Основной пайплайн машинного обучения:
  - формирование панельного датасета (SKU × клиент × месяц);
  - feature engineering (лаговые признаки, сезонность, промо-факторы);
  - обучение и сравнение моделей (Ridge, ElasticNet, RandomForest, HistGradientBoosting);
  - оценка качества (RMSE, MAE, R²);
  - решение обратной задачи: подбор минимальной скидки для заданного роста продаж.

---

### Данные
- **sales.csv** — данные по розничным продажам;
- **promo_sales.csv** — фактические промо-продажи;
- **promo_final.csv** — планы промо-акций и уровни скидок.

---

## Используемые технологии
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  


---

## Цель проекта
Количественно оценить влияние промо-акций и скидок на продажи, построить модель прогнозирования спроса и показать возможность персонализированного подбора скидок для достижения заданного роста продаж.
